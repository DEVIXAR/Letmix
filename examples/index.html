How it work

Є два типи роботи з текстом:
перебір символів 
$('#element').Letmix();
We do that we love to do
та біг по символах 
$('#element').Letmix('runner');
We do that we love to do

Options

Перебір символів

Amount of chars when is generation
>>> Letmix.amountIterations:Integer
Default: 3

$('#element').Letmix({
    amountIterations: 40
});
We do that we love to do
Color iteration
>>> Letmix.colorIteration:String
Default: CF3636
#123 => hex | 123 => hex | 123123 => hex | #123123 => hex | 123, 123, 123 => rgb | 123, 123, 123, 0.4 => rgba 

$('#element').Letmix({
    colorIteration: '#3f9'
});
We do that we love to do
List of chars
>>> Letmix.chars:Array< string >
Default: ['A', 'a', 'B', 'b', 'C', 'c', '1', '2', '3']

$('#element').Letmix({
    chars: ['1', '2', '3', '4', '5', '6']
});
We do that we love to do
Type iteration. How will be get char from list.

* random (get random item), line (get item by way), custom (get item by @customOrder)
>>> Letmix.typeIteration:String
default: 'random'

$('#element').Letmix({
    typeIteration: 'line'
});
We do that we love to do
Custom order list
>>> Letmix.customOrder:Array< Integer >
default: [2, 1, 0]

$('#element').Letmix({
    typeIteration: 'custom',
    customOrder: [1, 2, 3, 4, 0],
    chars: ['A', 'B', 'C', 'D', 'E']
});
We do that we love to do
Speed for one iteration. -> A -> B
>>> Letmix.speedIteration:Integer (ms)
default: 30

$('#element').Letmix({
    speedIteration: 100
});
We do that we love to do
Wait some time & run
>>> Letmix.wait:Integer (ms)
default: 0

$('#element').Letmix({
    wait: 1000 // Wait 1 sec. and start Letmix
});
We do that we love to do
Event start
>>> Letmix.start:Function( Function( Object ) )
default: 0

$('#element').Letmix({
    start: function(object){
        // object - you can use for some work
        // object.addClass('active');

        alert('Start');
    }
});
We do that we love to do
Event finish
>>> Letmix.end:Function( Function( Object ) )
default: 0

$('#element').Letmix({
    end: function(object){
        // object - you can use for next Letmix
        // object.Letmix(); or object.removeClass('active');

        alert('Finish');
    }
});

Examples


$('#element').Letmix({
    amountIterations: -1,
    speedIteration: 20,
    chars: ['_', ' '],
    type: 'line',
    colorIteration: '000',
    end: function (e) {
        e.Letmix('runner', {
            direction: 'left=>right',
            color: '0, 100, 200'
        });
        e.Letmix('runner', {
            color: '0, 200, 20'
        });
    }
});
We do that we love to do

$('#element').Letmix({
    amountIterations: -1,
    speedIteration: 20,
    chars: ['a', 'b', 'x', 'z'],
    typeIteration: 'line',
    type: 'random',
    colorIteration: '#32f732',
    end: function (e) {
        e.Letmix('runner', {
            direction: 'left=>right',
            color: '0, 100, 200',
            stopWhere: 'We do that'
        });
        e.Letmix('runner', {
            color: '0, 200, 20',
            stopWhere: 'we love to d'
        });
    }
});
